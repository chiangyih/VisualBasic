# 建置一個簡單的規格文件，描述一個軟體系統的需求和功能。
# 軟體規格文件 (Software Specification Document)
## 1. 簡介
### 1.1 目的: 讀取電腦記憶體使用率
## 2. 功能需求
### 2.1 記憶體使用率讀取
- 系統應能夠讀取並顯示當前電腦的記憶體使用率。
- 系統應能夠以百分比形式顯示記憶體使用率。
- 系統應能夠每每秒更新記憶體使用率數據。
- 記憶體使用率數據儘可能跟作業系統的工作管理員所顯示的數據一致。
### 2.2 使用者介面
- 系統應提供一個簡單的圖形使用者介面 (GUI) 來顯示記憶體使用率。
- GUI包含顯示記憶體使用率的主要視窗,一個啟動顯示使用率的按鈕，一個結束程式的按鈕
